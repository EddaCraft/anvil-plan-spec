# anvil-plan-spec

Templates and guidance for APS (Anvil Plan Spec), a lightweight format for planning and task authorization in AI-assisted development.

## Contents

- `Index_template.txt`: Non-executable plan index template.
- `leaf_template.txt`: Module/leaf template with task authorization rules.
- `simple_template.txt`: Feature-level template for small scopes.
- `docs/ai/`: Policies and prompting guidance for APS usage.
